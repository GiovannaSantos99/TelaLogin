# Tela de Login

O projeto foi feito com as seguintes tecnologias: Angular e Bootstrap.

**OBS: Para acessar o projeto é necessário ter o node js instalado.**

1º - Ir na pasta raiz do projeto e digitar o comando "ng serve".
2º - Para Visualizar no browser: http://localhost:4200/
